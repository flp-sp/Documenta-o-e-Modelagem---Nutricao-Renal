# NutriRim

## Objetivo
Controlar o consumo de alimentos e água de pacientes com problemas renais crônicos de acordo com o perfil individual.

## Escopo
* **Mapeamento:** Armazenamento do perfil clínico e restrições do paciente.
* **Monitoramento:** Registro de água e alimentos com cálculo automático de nutrientes.
* **Alertas:** Notificações visuais ao atingir 90% do limite diário estabelecido.
* **Sugestões:** Filtro de receitas culinárias adequadas à dieta do usuário.
* **Suporte:** Chatbot para dúvidas de alimentação e uso do aplicativo.
